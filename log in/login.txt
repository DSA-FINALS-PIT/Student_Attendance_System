VARIABLES USED IN LOGIN SYSTEM

DATABASE_FILE
- Stores the name/path of the text database file.

file
- Used when opening and reading/writing files.

accounts
- Dictionary storing all loaded account data.

line
- Represents one line from the text file.

data
- Stores split line data from the text file.

username
- Stores username input or loaded username.

password
- Stores password input or loaded password.

role
- Stores user role.

root
- Main Tkinter window.

username_entry
- Username input field in login form.

password_entry
- Password input field in login form.

create_window
- Window for creating accounts.

new_username
- Username input field for new account.

new_password
- Password input field for new account.

role_var
- Tkinter StringVar storing selected role.

forgot_window
- Window for forgot password feature.

forgot_username
- Username input field for password recovery.


FUNCTIONS USED

load_accounts()
- Loads accounts from TXT file.

save_account()
- Saves new account to TXT file.

login()
- Handles login process.

create_account()
- Opens create account window.

register_account()
- Saves newly created account.

forgot_password()
- Opens forgot password window.

recover_password()
- Retrieves forgotten password.
